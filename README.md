# Commandes-Powershell


### Les commandes PowerShell qui correspondent aux commandes Linux (shell bash) suivantes :
> cp = **Copy-Item**

> rm = **Remove-Item**

> cd = **Set-Location**
 
> mkdir = **New-Items**
 
> man = **help**
 
> history = **Get-History**

> alias = **New-Alias**

> cat = **Get-Content** 

> Ligne de commandes = Get-Alias
